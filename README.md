# First Website - Clases de Física y Matemática

Sitio web desarrollado como proyecto personal para ofrecer clases particulares de Física y Matemática.

El objetivo de este proyecto es practicar y consolidar conocimientos en desarrollo web frontend, construyendo una landing page real, responsive e interactiva, publicada online y vinculada a un caso de uso propio.

Además, el proyecto incluye un modelo inicial de base de datos en MySQL para organizar consultas, clases agendadas y pagos.

## Base de datos

El proyecto incluye una carpeta `database/` con un modelo inicial en MySQL para gestionar la actividad de clases particulares.

La base contempla:

- `consultas`: personas interesadas, materia, modalidad, disponibilidad y estado.
- `clases`: clases agendadas o realizadas, fecha, hora, duración, tema y precio.
- `pagos`: señas, pagos completos o saldos pendientes.

Archivos incluidos:

- `schema.sql`: estructura de la base de datos.
- `seed_demo.sql`: datos ficticios para pruebas.
- `queries.sql`: consultas útiles para agenda, pagos y estadísticas.

Los datos incluidos en el repositorio son ficticios y se usan únicamente como demostración.
